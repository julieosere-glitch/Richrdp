# Trading Analysis Platform

This is a web-based trading analysis platform that uses AI agents to analyze historical trade data and real-time news for making trading decisions on Forex, futures, and cryptocurrency markets.

## Features

- Real-time market data integration
- Historical trade data analysis
- News sentiment analysis for trading decisions
- AI-powered trading recommendations
- Visual dashboard for monitoring trading signals
- OANDA API integration for Forex data

## Setup

1. Install dependencies: `npm install`
2. Create a `.env` file with your OANDA API credentials
3. Run the application: `npm start`

## Technologies Used

- Node.js/Express for backend
- React for frontend
- OANDA API for market data
- News API for real-time news
- AI/ML models for analysis
